## ガイドライン

### 作成の進め方

Phase 1 → Phase 2 → Phase 3 → Phase 4 の順で進める。一度に完璧を目指さず、まず Phase 1（現状把握）を完成させることを優先する。

### 書き方のポイント

**自己分析**
- 事実ベースで記載する（「〜ができる」より「〜をした」）
- 数値化できるものは数値化する（人数、期間、改善率など）
- 強みは「他者と比較して優れている点」を意識する

**市場分析**
- 「感覚」ではなく「データ」で示す
- 複数の情報源から客観的データを収集する
- 情報源: 転職サービス（転職ドラフト、ビズリーチ等）、エージェント面談、スカウト・オファー実績

**ギャップ分析**
- ギャップは具体的に記述する（「リーダーシップ」ではなく「50人以上の組織での技術統制経験」）
- 現実的に埋められるギャップかを見極める

**目標設定**
- 測定可能な目標にする（年収、ポジション名、組織規模など）
- 複数のキャリアパスを持ち、1つに固執しない

**アクションプラン**
- 期限を明確に設定する（「いつか」ではなく「3ヶ月以内」）
- 小さく始められるアクションに分解する

### 見直しサイクル

| 頻度 | 内容 |
|-----|------|
| 月1回 | アクションプランの進捗確認 |
| 四半期 | 実績追加、市場動向の反映 |
| 年1回 | 全体の戦略見直し |

### 活用シーン

- **転職活動**: 職務経歴書・面接準備の元データとして
- **評価面談**: 自己評価・目標設定の材料として
- **学習計画**: ギャップを埋めるための学習ロードマップ策定
- **日常業務**: 「何をやる・やらない」の意思決定基準として
